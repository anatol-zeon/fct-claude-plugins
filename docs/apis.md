## Правила ведения

- Новая внешняя зависимость = запись здесь + ADR с альтернативами.
- Изменение контракта = обновление спецификации в `docs/api/` + запись в CHANGELOG.
- Отключили сервис → не удаляем строку, переносим в "Deprecated" с датой и причиной.
